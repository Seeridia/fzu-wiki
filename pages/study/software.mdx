# 正版化软件

福州大学会为在校学生提供正版化软件的使用权限。以下是一些常用软件的获取方式：

## WPS 365

可以享受云空间、模板、AI 等等同于 WPS 最高会员的权益。

1. 在[官网](https://www.wps.cn/)下载并安装 WPS
2. 在登录界面选择 “SSO 专属账号”
3. 输入 “fzu” 后输入学号密码等信息即可

4. 使用刚刚登录好的企业账号点击此 [链接](https://365.kdocs.cn/l/ck0SRgWknFzR) 逐个领取会员权益（如果不是企业账号会要求申请查看，请点击右上角头像按照上面的步骤再进行一次登录）

<Callout type="info">
   该领取链接有一定时效性，每年会在福州大学公众号上发布新的链接，请关注公众号获取最新信息。
</Callout>

## 腾讯会议 教育增强版

可享受不限时长、人数等会议增强权益。

1. 在[官网](https://meeting.tencent.com/)下载并安装腾讯会议。
2. 在登录界面选择 “SSO”。
3. 输入 “fzu” 后输入学号密码等信息即可。

## 其他软件

下列较少使用的软件可以在 [福州大学 网信办正版化软件管理平台](https://wxb.fzu.edu.cn/zbhrjgl.htm) 获取

- 银河麒麟操作系统
- 统信桌面操作系统
- WPS信创版本
- 中望CAD
- 中国卡伦特在线CAD
- 等

当然，许多软件也都能凭教育邮箱获取，关于教育邮箱可以参考 [教育邮箱](vpnmail.mdx) 页面，下面是列举的一些有提供教育邮箱获取的正版化软件：

- Microsoft 365（包括 Word、Excel、PowerPoint 等）
- JetBrains 系列（包括 IntelliJ IDEA、PyCharm 等）
- Autodesk 系列（包括 AutoCAD、Maya 等）
- Notion
- Figma
- 等
